# Accordion

An accordion component.

## Props

| Name     | Type            | Description          |
| -------- | --------------- | -------------------- |
| children | React.ReactNode | Accordion content    |
| title    | string          | Accordion title      |
| isOpen   | boolean         | Accordion open state |

## Example

```jsx
<Accordion
  title='Accordion Title'
  isOpen={true}
>
  <p>Accordion Content</p>
</Accordion>
```
