# Button

A button component.

## Props

| Name     | Type     | Description                                  |
| -------- | -------- | -------------------------------------------- |
| onClick  | function | Callback function when the button is clicked |
| children | node     | Node to display on the button                |

## Example

```jsx
<Button
  onClick={() => {}}
  children={<span>Click me</span>}
/>
```
